# Exodus UI

Единый интерфейс системы.

Он закрывает весь пользовательский цикл:
- ввод исходной задачи
- диалог для уточнения
- показ того, что система поняла (Intent)
- показ плана действий (Control)
- подтверждение запуска
- просмотр результата выполнения
- наблюдение состояния системы и прогресса
