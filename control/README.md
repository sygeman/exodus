# Exodus Control

Слой планирования и принятия решений.

Он берет Intent и формирует план выполнения:
- декомпозиция задачи
- выбор стратегии реализации
- распределение работы
- выбор инструментов и моделей из Infra
- определение зависимостей шагов

**Перед Execution Control обязан показать:**
- что система поняла
- что она собирается делать

И дождаться подтверждения пользователя.
